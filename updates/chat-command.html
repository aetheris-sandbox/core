Чат & Команды

9 августа 2025 Автор: Lelik

Чат

Добавлен чат, для общения с другими игроками:

  • При заходе / выходе игрока в чат выводится соответвующее сообщение
  • Чат можно активировать на букву T

Команды

Система команд, для активации откройте чат и начните сообщение с / :

  • /create [prop / car] [название] - создает объект перед игроком
  • /delete [prop / car] [кол-во] - удаляет последние созданные объекты / автомобили (только если внутри нет игрока)
  • /tp [имя_игрока] - телепортирует вас к выбранному игроку
  • Дополнительно: теперь у каждого игрока в паузе (если вы являетесь хостом) есть кнопка Kick, позволяющая выгнать игрока с сервера

Добавлено

Что добавлено еще:

  • Урон от падения
  • Копирование никнейма по нажатию на слот
  • Кнопки для выбора типов пропов в панели спавна
  • Теперь если с объектом можно взаимодействовать, слева будет панель указывающая на это
  • Интерфейс теперь можно скрывать
